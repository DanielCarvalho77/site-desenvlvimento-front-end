Front-end

Desenvolvimento front-end, é o termo técnico utilizado para a área do desenvolvimento de software, voltado a toda parte "de frnte" do sistema, ou seja, todas as telas e aquilo que o usuário vê, layout, template e tudo que a tela mostra basicamente.
É uma das áreas mais importantes para a experiência de usuário, sempre deve-se prezar por telas clean, ou limpas, sem poluição visual, facilitando a usabilidade do usuário.
Quanto mais agradável o front maior será a adesão do sistema, mas isso não significa tudo, o sustema tem que ser funcional, e isso vamos abordar no próximo texto.

Back-end

No texto acima abordamos a parte do desenvolvimento que entrega tudo o que usuário pode ver, bem, aqui a coisa é diferente, no back-end nada fica amostra, você pode entender que as vezes quando seu usuário não loga, ou quando você carrega uma informação salva, então quem faz esse trabalho é o back-end.
É o código que conecta a internet com o banco de dados, gerencia as conexões dos usuários e alimenta a aplicação web. O backend trabalha em conjunto com o frontend para entregar o produto para o usuário final.
